About Baking Master

Hi, hello! How are you? My name is Karol and I'm an author of this web app.This is my first web application runned by Python, created in AWS Cloud9 and it is also deployed on Heroku Pages. This project was created to show my current skills and what I have learned while studying my course from Code Institute.

I hope you'll enjoy time spent here and you will find some fantastic recipes that will make you a Baking Master.
